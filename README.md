# activity-report-maker

## Install

1. clone

```bash
git clone https://github.com/jeremN/activity-report-maker.git
# or
git clone git@github.com:jeremN/activity-report-maker.git
```

2. install node dependencies

```bash
npm i
```

## Commands

```bash
# run dev environment
npm run dev
# build
npm run build
# build preview
npm run preview
# deploy preview
npm run predeploy
# deploy on github page
npm run deploy
```
